PRODUCTOS DE TEMPORADA

  • Llavero de pudín de arroz
    Este pequeño llavero de patata Ricespud rebosa de suavidad y ternura. Está hecho de tela beige muy suave y se completa con una práctica cadena de bolas negra para colgarlo de las llaves, los bolsos o donde quieras. ¡Puedes llevar este llavero de Ricespud a cualquier parte! El compañero perfecto para las aventuras en movimiento.
  • Honguito claro 
    Hecho de tela marrón muy suave y esponjosa, con pequeñas astas de fieltro, una barriguita blanca suave, cola de pompón blanco y una pequeña nariz de pompón rojo, ¡como Rodolfo! Este reno Ricespud está lleno hasta el borde para esos abrazos extra regordetes y está listo para ser el regalo de Navidad perfecto para ti o tus seres queridos esta Navidad.
  • Rabanito 
    ¡Dale la bienvenida a este gigante de peluche Ricebao, enorme y blando, a tu hogar! ¿No se ven deliciosos? ¡Chupbby Ricebao se ha vuelto aún más blando! Presentamos nuestro nuevo peluche gigante Ricebao, el amigo cojín perfecto para colocar en cualquier cama o sofá para hacerte compañía todo el día.
  • Mini pudín reno 
    Este mini llavero Ricemogu en forma de hongo está lleno de suavidad y ternura. Hecho de tela esponjosa blanca y roja muy suave y completo con una práctica cadena de bolas negras para enganchar a las llaves, bolsos o cualquier lugar que desees: ¡puedes llevar este llavero Ricemogu a cualquier parte! El pequeño compañero perfecto para aventuras en movimiento.
  • Pin de arveja 
    ¡Echa un vistazo a nuestra nueva y brillante colección de pins de esmalte duro! Este adorable pin de col de Bruselas Riceprout es el pin perfecto para rellenar una media navideña o para usar durante todo el año. ¡Por qué no decirle a todos que te encantan los brotes!
  • Postre navideño 
    Está hecho de tela suave y esponjosa, con pequeños brazos, botones y nariz de fieltro y está relleno hasta el borde para esos abrazos extra regordetes. Además, su práctica correa de velcro y su nuevo llavero de tela significan que puedes colocarlos o colgarlos donde quieras.
  • Bola de arroz gigante 
    Riceparsnip es la verdura de raíz más generosa de Ricefarm, ¡siempre les da regalos a sus compañeros Ricemonsters incluso cuando no es Navidad! Esta pequeña verdura es el amigo perfecto para hacerte sonreír todo el año. Con un cuerpo de color crema, mejillas anaranjadas y cabello verde, es la adición perfecta a tu colección de peluches.
  • Pudin muñeco de nieve  
    Deja que este adorable amigo regordete te caliente el corazón esta Navidad. ¿No se ven dulces? Ricemaspud tiene un cuerpo marrón, cabello esponjoso de color crema y una linda cara bordada. ¡Usa su correa colgante extraíble para agregarlos a tu árbol de Navidad!
  • Llavero de pudin 
    Este pequeño llavero de patata Ricespud rebosa de suavidad y ternura. Está hecho de tela beige muy suave y se completa con una práctica cadena de bolas negra para colgarlo de las llaves, los bolsos o donde quieras. ¡Puedes llevar este llavero de Ricespud a cualquier parte!
